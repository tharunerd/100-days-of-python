# 🚀 How to Run Any Project

### 1️⃣ Install Python

Make sure you have **Python 3.10+** installed.

### 2️⃣ Clone the Repository

```bash
git clone https://github.com/tharunerd/100-days-of-python.git
```

### 3️⃣ Open Any Day’s Project

```bash
cd 100-days-of-python/day030-password-manager
```

### 4️⃣ Run the Project

```bash
python main.py
```

🏝️ Build your first **text-based adventure game** using `if/elif/else` conditions.
