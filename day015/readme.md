# ☕ Day 15 – Coffee Machine Project (Python)

## 🚀 Overview

On **Day 15** of my Python learning journey, I built a **Coffee Machine simulation** using Python.
This project helped me understand how to apply programming concepts to a real-world scenario by simulating the working of a coffee vending machine.

---

## 🎯 Project Objective

To create a console-based coffee machine that:

* Accepts user input for drink selection
* Processes coin payments
* Checks resource availability
* Serves coffee and updates resources

---

## 🧠 Concepts Used

* Functions
* Conditional Statements (`if/elif/else`)
* Loops (`while`)
* Dictionaries
* User Input Handling
* Basic Logic Building

---

## ⚙️ Features

* ☕ Menu with multiple drinks (Espresso, Latte, Cappuccino)
* 💰 Coin-based payment system
* 📊 Resource management (water, milk, coffee)
* 🧾 Report generation for remaining resources
* ❌ Handles insufficient resources or payment

---

## 💻 How It Works

1. User selects a drink
2. Machine checks if resources are sufficient
3. User inserts coins
4. Machine validates payment
5. Coffee is served ☕
6. Resources are updated

---

## 📂 Files Included

* `coffee_machine.py` – Main program file

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone the repository
3. Run the program:

```bash
python coffee_machine.py
```
